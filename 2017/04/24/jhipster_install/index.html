JHipster란?

Jhipster는 Yeoman진영에서 개발된 Angularjs + Spring boot기반의 어플리케이션 프로젝트이다. 클라이언트, 서버측 기술들을 일일이 나열하기 힘들정도로 방대하게 제공해주고 유용한 툴이나 제공되는 개발환경 세팅이 잘되어있다.

자세한 설명은 jhipster 사이트를 참고!
